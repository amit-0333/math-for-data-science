# Math for Data Science

A structured collection of notes and resources covering the core mathematical foundations required for data science — statistics, probability, and linear algebra — compiled while following the **CampusX DSMP 1.0 (Data Science Mentorship Program)** curriculum.

## 📂 Repository Structure

| Folder | Description |
|---|---|
| `01_Descriptive_Statistics` | Notes on measures of central tendency, dispersion, distributions, and other descriptive statistics concepts. |
| `02_Probability & Distributions` | Probability theory basics and common probability distributions. |
| `03_Inferential_Statistics` | Confidence intervals, Central Limit Theorem, hypothesis testing, and related inferential concepts. |
| `04_Linear_Algebra` | Linear algebra fundamentals including linear transformations. |
| `raw-session-notes` | Rough/unedited notes taken during live sessions, kept for reference before being cleaned up and merged into the main topic folders. |

## 📌 About

These notes are a work in progress and are being added to on an ongoing basis. The goal is to build a solid, organized reference for the math that underpins data science and machine learning.

- Course reference: **CampusX DSMP 1.0**
- Any topics not yet covered here will be added as they are studied.
- Folders may be reorganized or renamed over time as content is refined.

## 🚧 Status

This repository is actively maintained. Some sections are complete, while others are still being written up from raw session notes. Leftover/pending topics will be covered and added as and when needed.

This is primarily a personal learning repository, but suggestions or corrections are welcome via issues or pull requests.
